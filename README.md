## 需求整理
- “管理员”注册入口，默认有一个超级管理员，暂时不做找回密码功能
- 要统计各个资源的下载次数，资源类型包括：文本、pdf、视频、zip
- 资源分类，默认2级分类，首页展示
- “教师身份”可以上传自己的资源
- 资源要能按照上传人员、分类、下载数量维度排序
